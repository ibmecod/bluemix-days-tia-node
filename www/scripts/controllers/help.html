Twitter Influencer Analyzer

This app gathers information from various sources regarding Twitter users. Enjoy!

Fork this project on GitHub.

Home page

The home page displays a list of all the twitter users you are analyzing.

You can remove a user from the list by pressing the icon.

Click on the icon to analyze this user.

Click on the icon to view the user's Twitter page.

The badged number, like 00 is the user's Klout score.

At the top of the page, you can enter a new Twitter user screen name to analyze.

Messages page

This page displays error and informational messages.
